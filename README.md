#  EDUCO Dashboard UI

This is a responsive educational dashboard UI built with **React** and **Tailwind CSS** as part of the *Round 1* task.

##  Features

- Clean sidebar navigation
- Dashboard with:
  - New Courses
  - Hours Activity Chart
  - Daily Schedule
  - Calendar & Assignments
  - Ongoing Courses
- Fully responsive layout
- Modern aesthetic with subtle animations

##  Tech Stack

- React.js
- Tailwind CSS
- React Icons
- Google Fonts: *Plus Jakarta Sans*
